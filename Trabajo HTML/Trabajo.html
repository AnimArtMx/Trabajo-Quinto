📚 Lección 1: ¿Qué es HTML?

HTML significa Lenguaje de Marcado de HiperTexto. Es como el esqueleto de una página web - ¡le da estructura!

🎯 Dato Curioso: HTML fue creado en 1991 por Tim Berners-Lee. ¡Tiene más de 30 años!

¿Cómo funciona HTML?

HTML usa etiquetas para decirle al navegador qué mostrar. Las etiquetas se escriben entre símbolos de "menor que" y "mayor que":

<etiqueta>contenido aquí</etiqueta>
Ejemplo:
<h1>¡Hola Mundo!</h1>

Esto muestra: ¡Hola Mundo!

💡 Para Recordar:
  • Casi todas las etiquetas tienen apertura <etiqueta> y cierre </etiqueta>
  • El contenido va en el medio
  • HTML no es sensible a mayúsculas, ¡pero es mejor escribir en minúsculas!

🏷️ Lección 2: Etiquetas Básicas de HTML

Estructura básica de una página HTML:

<!DOCTYPE html> <html> <head> <title>Mi Página</title> </head> <body> ¡Aquí va el contenido! </body> </html>

Etiquetas que debes conocer:

1. Encabezados (Títulos):
<h1>Título Grande</h1> <h2>Título Mediano</h2> <h3>Título Pequeño</h3>

Resultado:

Título Grande

Título Mediano

Título Pequeño

2. Párrafos:
<p>Este es un párrafo. Se usa para escribir texto normal.</p>
3. Listas:
<ul> <li>Primer elemento</li> <li>Segundo elemento</li> </ul>

Resultado:

  • Primer elemento
  • Segundo elemento
4. Enlaces:
<a href="https://www.google.com">Ir a Google</a>
5. Imágenes:
<img src="foto.jpg" alt="Descripción de la imagen">

Nota: Esta etiqueta no necesita cierre

🎨 Lección 3: ¿Qué es CSS?

CSS significa Hojas de Estilo en Cascada. ¡Es lo que hace que las páginas web se vean bonitas!

🎯 Piensa así: Si HTML es el esqueleto, CSS es la ropa, el peinado y el maquillaje de tu página web.

¿Cómo se escribe CSS?

CSS tiene tres partes: selector, propiedad y valor

selector { propiedad: valor; }
Ejemplo:
p { color: blue; font-size: 20px; }

Esto hace que todos los párrafos <p> sean azules y de tamaño 20 píxeles.

¿Dónde se pone el CSS?

Puedes poner CSS directamente en las etiquetas con style="":

<h1 style="color: purple;">Título Morado</h1>

🌈 Lección 4: Colores y Estilos Divertidos

1. Colores

Puedes usar nombres de colores en inglés o códigos especiales:

h1 { color: red; /* Color del texto */ background-color: yellow; /* Color de fondo */ }
¡Así se ve!

2. Tamaños

p { font-size: 24px; /* Tamaño de letra */ width: 300px; /* Ancho */ height: 100px; /* Alto */ }

3. Bordes y Espacios

div { border: 3px solid blue; /* Borde */ padding: 20px; /* Espacio interno */ margin: 10px; /* Espacio externo */ border-radius: 15px; /* Esquinas redondeadas */ }
¡Mira qué bonito se ve con bordes redondeados!

4. Texto

p { text-align: center; /* Alineación */ font-weight: bold; /* Negrita */ font-style: italic; /* Cursiva */ text-decoration: underline; /* Subrayado */ }
